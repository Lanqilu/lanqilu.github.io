格式化字符串

字符串提供了 formatted() 方法（JDK 15）和 format() 静态方法，可以传入其他参数，替换占位符，然后生成新的字符串：

有几个占位符，后面就传入几个参数。参数类型要和占位符一致。我们经常用这个方法来格式化信息。常用的占位符有：

  • %s：显示字符串；
  • %d：显示整数；
  • %x：显示十六进制整数；
  • %f：显示浮点数。

占位符还可以带格式，例如 %.2f 表示显示两位小数。如果你不确定用啥占位符，那就始终用 %s ，因为 %s 可以显示任何数据类型。要查看完整的格式化语法，请参考JDK文档。
